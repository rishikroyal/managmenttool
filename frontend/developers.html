🔐 Authentication

POST /auth/login

Exchange user credentials for a secure JWT bearer token.

curl --location 'https://managmenttool-53ja.onrender.com/auth/login' \ --header 'Content-Type: application/json' \ --data-raw '{ "email": "admin@gmail.com", "password": "123456" }'
GET /me

Retrieve profile information for the currently authenticated user.

curl --location 'https://managmenttool-53ja.onrender.com/me' \ --header 'Authorization: Bearer YOUR_TOKEN_HERE'

📁 Projects

POST /projects/

Initialize a new project. The creator is automatically assigned as the Project Admin.

curl --location 'https://managmenttool-53ja.onrender.com/projects/' \ --header 'Authorization: Bearer YOUR_TOKEN_HERE' \ --header 'Content-Type: application/json' \ --data '{"name":"Team Task Manager","description":"Main assignment project"}'

✅ Tasks & AI

POST /tasks/

Create a task and assign it to an employee within a project.

curl --location 'https://managmenttool-53ja.onrender.com/tasks/' \ --header 'Authorization: Bearer YOUR_TOKEN_HERE' \ --header 'Content-Type: application/json' \ --data '{ "title": "Design Dashboard", "description": "Create analytics dashboard UI", "priority": "high", "assigned_to": 1, "project_id": 1 }'
GET /tasks/suggest-assignee

Utilize Groq Llama 3 AI to analyze workload and suggest the best employee for a task.

curl --location 'https://managmenttool-53ja.onrender.com/tasks/suggest-assignee' \ --header 'Authorization: Bearer YOUR_TOKEN_HERE' \ --header 'Content-Type: application/json' \ --data '{ "task_description": "Create a high-performance analytics dashboard" }'
